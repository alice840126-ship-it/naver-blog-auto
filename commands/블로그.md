# 블로그 생성

네이버 SEO 최적화 블로그 포스트를 자동으로 생성합니다.
SERP 분석을 기반으로 블로그 글과 섹션별 이미지를 생성하고 바탕화면 블로그자동화 폴더에 저장합니다.

## 파이프라인

naver-analyzer + serp-analyzer (병렬) → blog-writer-naver → blog-image → blog-saver

## 사용법

/블로그 [키워드]

## 예시

/블로그 클로드 코드 활용법
/블로그 챗GPT vs 클로드 비교
/블로그 네이버 블로그 상위노출 방법

## 동작

다음 파일을 읽고 안에 적힌 파이프라인 지시대로 실행한다:

__AGENTS__/orchestration-blog.md

키워드: $ARGUMENTS

## 결과

- 바탕화면 > 블로그자동화 폴더에 마크다운 파일 저장
- 커버 이미지 1장 + H2 섹션당 이미지 1장 생성
- 블로그자동화/images/ 폴더에 이미지 저장
- 네이버 블로그에 수동 업로드하면 완료

## 주의사항

- 자동 발행 없음 — 네이버 블로그에서 직접 업로드 필요
- 이미지 생성에 Gemini API Key 필요 (~/.claude/.env에 GEMINI_API_KEY 설정)
- 예상 소요 시간: 약 15-20분 (이미지 포함)
- API Key 없으면 글만 생성, 이미지 스킵
